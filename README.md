# AwesomePOS
#vue实战小项目-快餐店收银系统;
##vue技术栈
- vue   webpack   Element-ui   Axios   vue-router

## 脚手架
````
$ npm install vue-cli -g
````
## 采用webpack模板，初始化项目
````
$ vue webpack AwesomePos
````
## Element-ui
````
$npm install element-ui --save
````
##Axios
````
$ npm install axois --save
````
##项目打包
````
$ npm run build
````
